## 分析图表

### 头部

- 标题「Analytics」+ 刷新按钮
- 时间范围选择器（开始日期 ~ 结束日期）

### 图表

**活动时间线**（全宽堆叠面积图）：
- 按类别分色：login、user、session、tool、skill
- 图例可点击隐藏/显示各类别
- X 轴：时间，Y 轴：操作次数

**热门技能**（左半，水平条形图）：
- Y 轴：技能名称（最大宽度 140px）
- X 轴：调用次数
- 紫色条形
- 高度根据技能数量动态调整

**会话活动**（右半，垂直条形图）：
- X 轴：日期，Y 轴：会话数
- 蓝色条形
- 固定高度 220px

## 审计日志

### 筛选控件

```
┌──────────────────────────────────────────────────┐
│ 📅 开始日期 ~ 结束日期                              │
│                                                  │
│ FILTER BY: [All ▼] : [搜索目标...]                │

```

**筛选类别：**

| 类别 | 筛选字段 | 搜索占位符 |
|------|---------|-----------|
| All | target | Search target |
| Login | target | Search target |
| User | target | Search target |
| Session | session_id | Search session ID |
| Skill | target | Search target |
| Tool | target | Search target |
| Actor（仅管理员）| actor | Search actor name |

### 两栏布局

**左栏：条目列表**

每条审计记录显示：

```
┌──────────────────────────────────────────┐
│▎LOGIN.SUCCESS              2 minutes ago │  ← 左侧彩色边框 + 操作名(大写) + 相对时间
│ admin logged in successfully             │  ← 描述
│ ▸ Details                                │  ← 可展开 JSON 详情
└──────────────────────────────────────────┘
```

**操作颜色编码：**
- 🟢 绿色：login.success、user.created
- 🔴 红色：login.failed、user.deleted、user.apikey_revoked
- 🟡 黄色：user.role_changed、user.password_reset
- 🔵 青色：tool.* 操作
- 🟣 紫色：skill.* 操作
- 蓝色：其他操作

底部有「Load more」按钮加载更多（每批 50 条）。

**右栏：活动图表**

与分析页面相同的堆叠面积图，展示审计活动趋势（独立数据源，最多 200 条）。

## 文件浏览器

目录式文件浏览界面，提供不同于表格视图的文件管理方式。

### 布局

```
┌──────────────────────────────────────────────────────────────────────────┐
│ / ▸ home ▸ user                          [Upload] [...path...] [Go]     │
│ 3 dirs, 12 files    🔍 搜索...                                          │
├────────────────────────────┬─────────────────────────────────────────────┤
│  文件列表（可拖拽宽度）      │  预览面板                                    │
│ ┌────────────────────────┐ │ ┌─────────────────────────────────────────┐ │
│ │ NAME ▲  SIZE  MODIFIED │ │ │ /home/user/app.py  ✨  📋  1.2KB  📥  │ │
│ ├────────────────────────┤ │ ├─────────────────────────────────────────┤ │
│ │ 📁 ..                  │ │ │  1 │ import os                         │ │
│ │ 📁 src                 │ │ │  2 │ import sys                        │ │
│ │▎📄 app.py     1.2KB   │ │ │  3 │                                   │ │
│ │ 📄 config.json 256B   │ │ │  4 │ def main():                       │ │
│ │ 🖼 logo.png   48KB    │ │ │  5 │     print("hello")                │ │
│ └────────────────────────┘ │ │    │ ...                               │ │
│                            │ └─────────────────────────────────────────┘ │
└────────────────────────────┴─────────────────────────────────────────────┘
```

### 顶部工具栏

- **面包屑导航**：`/` → 各级目录，点击跳转
- **上传按钮**（仅管理员）：上传文件到当前目录，上传时显示进度条
- **路径输入框**：手动输入绝对路径，按 Enter 或点击「Go」跳转

### 文件列表

- 表头支持排序：NAME、SIZE、MODIFIED（点击切换升序/降序）
- 图标区分类型：📁 目录、📄 文本、🖼 图片、📝 代码
- 点击目录进入子目录，点击文件在右侧预览
- 当前选中文件显示蓝色左边框高亮

### 预览面板

**头部**：
- 文件绝对路径（等宽字体）
- ✨ **Ask for Priva** 图标按钮（仅文本文件显示，见下方说明）
- 📋 复制路径按钮
- 文件大小
- 📥 下载按钮

**预览区域（按文件类型自动渲染）**：

| 文件类型 | 预览方式 |
|---------|---------|
| 代码/文本文件 | 语法高亮 + 行号 + 等宽字体 |
| PDF | 嵌入式 PDF 查看器 |
| 图片 (PNG/JPG/GIF/SVG 等) | 图片预览 |
| 二进制/不支持 | 显示文件大小 + 提示文字 |

### Ask for Priva（询问 Priva）

在文件浏览器中可以快速向 Priva 提问，请求帮助分析或改进文件内容。有两种触发方式：

**方式一：选中文本**

1. 在预览面板中用鼠标选中一段代码/文本
2. 选中后在鼠标位置弹出 ✨「询问 Priva」气泡按钮
3. 点击气泡 → 弹出评论弹窗，显示：
   - 文件路径 + 行号范围
   - 选中内容的代码预览（含 ±2 行上下文，选中部分高亮）
   - 评论输入框
4. 输入评论后点击「发送」

**方式二：整个文件**

1. 点击预览面板头部文件名旁的 ✨ 图标按钮
2. 弹出评论弹窗，选中内容为整个文件（L1 到最后一行）
3. 预览显示文件前几行 + 省略提示
4. 输入评论后点击「发送」

**发送后的行为**：

- 自动切换到 Priva 聊天标签页
- 开启新会话，权限模式设为 **Plan**
- 聊天输入框中显示**文件引用卡片**（青色左边框，显示文件路径、行号范围、代码片段）
- 用户评论填入输入框文本区
- **不会自动发送**，用户可以预览和修改后手动发送
- 发送后 Agent 会先读取完整文件，再结合上下文分析选中部分

**文件引用卡片**：

- 显示在聊天输入框的文本区上方
- 点击 × 可取消文件引用
- 在空输入框中按 Backspace 也可取消
- 发送后的消息气泡中会保留文件引用卡片（折叠状态，可展开查看代码）
- 重新加载页面或回顾历史会话时，文件引用卡片仍然正确渲染

## 用户管理（仅管理员）

### 两栏布局

**左栏：用户表格（50%）**

```
┌──────────────────────────────────────────────────┐
│ User Management                        [+ 创建]  │
│ 🔍 搜索用户...      [ALL] [ADMIN] [USER]          │
├──────────┬──────┬─────┬─────────┬────────────────┤
│ Username │ Role │ Key │ Created │ Actions        │
├──────────┼──────┼─────┼─────────┼────────────────┤
│▎admin    │ADMIN │ 👁  │ 03/01   │ [✏] [🗑]       │
│ user1    │USER  │ 👁  │ 03/15   │ [✏] [🗑]       │
│ user2    │USER  │ 👁  │ 03/20   │ [✏] [🗑]       │
└──────────┴──────┴─────┴─────────┴────────────────┘
```

- 搜索：按用户名过滤
- 角色筛选：ALL / ADMIN / USER 按钮切换
- API Key 列：👁 悬浮显示 Key 值 + 复制按钮
- 操作：✏ 编辑（打开抽屉）、🗑 删除（需确认，不能删除自己）
- 点击行：打开右侧 Inspect 面板

**右栏：用户详情 Inspect 面板（50%）**

```
┌──────────────────────────────────────────┐
│ admin                            ADMIN   │
│                                          │
│ [Skills]  [MCP]  [Scheduled ⏳]           │
├──────────────────────────────────────────┤
│ pdf           PDF 创建和处理      GLOBAL  │
│ docx          Word 文档操作       GLOBAL  │
│ my-skill      自定义技能         PROJECT  │
└──────────────────────────────────────────┘
```

- **Skills 标签**：该用户的技能列表（名称、描述、级别标签），可删除
- **MCP 标签**：该用户的 MCP 服务器列表（名称、URL、类型、级别标签），可删除
- **Scheduled 标签**：即将推出
