    │                  ▼
    │              创建成功，自动登录 ────────┐

    └── 已初始化 ──→ 显示「登录」页面          │
                       │                      │
                       ▼                      │

                       │                      │
                       ▼                      │
                   点击「Sign In」             │
                       │                      │

## 创建管理员账号

### 页面说明

首次访问 Priva 时自动显示此页面。居中显示，宽度 360px。

```
┌──────────────────────────────────────┐
│              🤖                       │
│          Set up Priva                │
│     Create your admin account         │
│                                      │
│  Username                             │
│  ┌──────────────────────────────┐    │
│  │ admin                         │    │  ← 默认值，可修改
│  └──────────────────────────────┘    │
│                                      │
│  Password                             │
│  ┌──────────────────────────────┐    │
│  │ ●●●●●●●●                     │    │
│  └──────────────────────────────┘    │
│                                      │
│  Confirm Password                     │
│  ┌──────────────────────────────┐    │
│  │ ●●●●●●●●                     │    │
│  └──────────────────────────────┘    │
│                                      │
│  ┌──────────────────────────────┐    │
│  │     Create Admin Account      │    │
│  └──────────────────────────────┘    │
└──────────────────────────────────────┘
```

### 字段说明

| 字段 | 要求 | 说明 |
|------|------|------|
| Username | 必填，不能为空 | 默认 `admin`，可自定义 |
| Password | 必填，不能为空 | 管理员密码 |
| Confirm Password | 必须与 Password 一致 | 防止输入错误 |

### 验证提示

- `Username is required` — 用户名为空
- `Password is required` — 密码为空
- `Passwords do not match` — 两次密码不一致
- `Setup failed` — 服务器错误

### 提交行为

1. 按钮文字变为 `Creating...`
2. 成功后自动登录（无需再进入登录页）
3. 系统为管理员账号准备工作空间和内置技能
4. 直接进入主界面

## 设置向导（LLM 配置）

### 触发条件

- 登录成功后，系统检查是否已配置 LLM 提供商
- 如果**未配置**，自动弹出设置向导弹窗
- 如果之前点过「Skip for now」，24 小时内不会再弹出

### 弹窗样式

- 居中显示，宽度 480px
- 半透明遮罩背景（带模糊效果）
- 弹出动画：缩放淡入（200ms）
- 顶部进度指示器：3 个圆点 `● ○ ○` 表示当前步骤

---

### Step 1/3 — 连接配置

```
┌──────────────────────────────────────────────────┐
│  Set up Priva                          ● ○ ○    │
│                                                  │
│  Enter your API base URL and                     │
│  authentication token.                           │
│                                                  │
│  BASE URL                                        │
│  ┌────────────────────────────────────────────┐  │
│  │ http://your-api-server:port/               │  │  ← 自动聚焦
│  └────────────────────────────────────────────┘  │
│                                                  │
│  AUTH TOKEN                                      │
│  ┌────────────────────────────────────────┬──┐  │
│  │ sk-●●●●●●●●●●●●●●●                    │👁│  │  ← 点击眼睛切换显示
│  └────────────────────────────────────────┴──┘  │
│                                                  │
│  ┌── 🟢 Connected — 12 model(s) found ───────┐  │  ← 自动测试结果
│  └────────────────────────────────────────────┘  │
│                                                  │
│  Skip for now                         [Next >]   │
└──────────────────────────────────────────────────┘
```

#### 操作说明

1. **填写 Base URL**：你的 AI 模型 API 服务器地址
2. **填写 Auth Token**：API 认证密钥
   - 右侧 👁 眼睛图标切换明文/隐藏
3. **自动连接测试**：两个字段都填写后，系统自动测试（800ms 延迟）

#### 连接状态

```
测试中：
┌── 🟡 Connecting... ──────────────────────────────┐
└──────────────────────────────────────────────────┘

成功：
┌── 🟢 Connected — 12 model(s) found ─────────────┐
└──────────────────────────────────────────────────┘

失败：
┌── 🔴 Connection failed: Network error ──────────┐
└──────────────────────────────────────────────────┘
```

- 状态框左侧有 2px 彩色边框（黄/绿/红）
- 连接成功后「Next」按钮变为蓝色可点击
- 连接失败时检查 URL 和 Token 是否正确

#### 跳过设置

- 点击左下角「Skip for now」可跳过
- 跳过后 24 小时内不会再次弹出
- 之后可在「设置 → LLM Provider」中手动配置

### 后续修改配置

如需修改 LLM 配置，不需要重新走设置向导：

1. 点击顶部导航栏右侧 ⚙ 齿轮图标
2. 在「LLM Provider」区域修改 Base URL 或 Auth Token
3. 点击 Save
4. 重新选择模型

## 常见问题

### 连接测试一直显示 "Connecting..."

- 检查 Base URL 是否正确（包含协议 `http://` 或 `https://`）
- 检查网络是否可达
- 确认 API 服务器已启动

### 连接成功但模型列表为空

- Auth Token 可能没有模型访问权限
- 确认 API 端点支持模型列表查询

### 跳过设置后如何重新配置

- 点击导航栏右侧 ⚙ 齿轮图标进入设置
- 在「LLM Provider」中填写配置
- 或等待 24 小时后系统自动再次弹出向导

### 忘记管理员密码

- 需要联系系统管理员重置
- 管理员可以在管理面板中为用户重置密码
